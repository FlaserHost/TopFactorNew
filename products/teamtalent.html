FAQ

Is it possible to analyze the effectiveness of job sites for a particular vacancy?

Yes, the effectiveness can be monitored.

For example, the report on the recruitment funnel shows which channels candidates who have passed one or more stages of selection come from.

There is also a separate report that identifies which channels generate the most applicants and who downloads the most of them. This report provides a flexible configuration of various filters: by vacancy, direction, date, and recruiter.

Is it possible to see at what stage the candidate was rejected (the history of interaction with him)?

Yes, it's possible. There is a separate report for analyzing failed candidates throughout the recruitment funnel. You can also see in the card of a particular candidate at which stage of the recruitment funnel a candidate was rejected.

How is the storage and processing of personal data implemented in the system?

PD is stored on the 1C platform side in accordance with the requirements of Federal Law. The system for working with personal data provides mechanisms for collecting consents and storing statuses on them, separating access using a role model and reports on all activities in the system. The data storage period for the candidate is set in the settings by the user. It is possible to set up several options for selecting by the candidate. Upon expiration of the storage period or refusal of the candidate, the data is depersonalized.

Do candidates give their consent to the processing of personal data to store their data?

Yes, for the storage of personal data, the system provides a mechanism for collecting consents to the processing of personal data. This mechanism is universal and is set up by the system administrator.

Is it possible to search for candidates in the system using certain criteria?

Yes, you will create a list of necessary candidates as easily as possible thanks to the flexibly configurable filters.

All fields in a candidate card can be search filters:
- characteristics,
- previous jobs,
- age,
- vacancy,
- direction, etc.

An informal search is possible, for example, by the presence of some words in the candidate's downloaded resume.

It is also easy to work with settings of a list of vacancies.

The filters you set up can be saved and used in future work. You can use complex filtering structures: for example, grouping filters using logical operators AND, OR, NOT.

Is it possible to set up an individual recruitment funnel?

Yes, the individual recruitment funnel is configured by our specialists as an additional project. Also, the administrator or the hiring manager can independently set up the recruitment funnel for the business processes of the company.

Is it possible to assign vacancies to recruiters taking into account the complexity or the department where the employee is required?

Yes, the head of recruitment can appoint a responsible recruiter for a vacancy and change it at any time. When a recruiter changes, all candidate data is saved.

Is it possible to edit the reasons for the rejection?

Yes, it is configurable in user mode.

Is it possible to monitor and redistribute the workload of recruiters?

Yes, you can track statistics on the active actions of recruiters in the system. To quickly redistribute the workload between recruiters, track statistics on vacancies in which recruiters are responsible, and reassign responsible recruiters if necessary.

Which messengers can be used to communicate with candidates directly from the system in your ut-of-the-box solution?

Communicate with candidates directly in the system via email and popular messengers:
- Telegram;
- WhatsApp;
- Viber;
- VKontakte.

Communication via WhatsApp requires to have a business account and an additional fee for providing the API by a separate service.

Is the online survey service available in the out-of-the-box solution?

Yes. The online candidate survey service, aimed at evaluating the candidate's personal qualities and professional competencies, is available in the out-of-the-box solution.

What are the technical requirements when installing an On-Premise solution?

The minimum system requirements are similar to the system requirements for the platform 1C:Enterprise 8". More detailed requirements should be clarified with your system administrators - they depend on the number of users connected at the same time, the volume of records, the file storage policy, etc.

Can the client customize the notification texts himself and add variables to templates and notifications?

Yes, you can flexibly customize notifications to suit your needs. The settings are available in the action templates.

It's possible:
- to record notification texts for both candidates and customers;
- to add tags for personalized sending;
- to add attachments downloaded from the file system and automatically generated inside the system, for example, an offer or referrals for medical examinations.

Does an invitation to the video interview come to the candidate only by mail? Is it possible to send an invitation via WhatsApp or Telegram?

You can flexibly customize all notifications in the system to suit your needs.

Sending an invitation to a video interview is configured as one of the stages of the selection funnel, which also has the ability to configure notifications. Notifications can be sent via email, any messenger and SMS.
